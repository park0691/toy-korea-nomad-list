## 1. 문제 정의

한국에서 디지털 노마드로 생활하고 싶은 사람들을 위한 체계적인 정보 플랫폼이 현재 존재하지 않는다.

### 현재 상황

- Nomads.com 등 글로벌 플랫폼은 한국 도시 데이터가 부실하고 원화 기준 생활비 정보가 없음
- 국내 커뮤니티(네이버 카페, 유튜브 등)에는 파편화된 후기가 산재하여 비교·탐색이 불가능
- 도시별 코워킹스페이스, 카페 WiFi, 대중교통 정보 등 노마드 특화 지표를 통합 제공하는 서비스 부재

### 해결하고자 하는 핵심 문제

- "어느 도시에서 노마드 생활을 시작할까?" — 비교 가능한 정보가 없어 결정이 어렵다
- 분산된 정보를 직접 수집·비교해야 하는 높은 탐색 비용
- 한국 특화 지표(카페 밀도, 전셋값 등)를 다루는 서비스의 부재

## 2. 타겟 사용자

### 2.1 주요 타겟

디지털 노마드로 살고 싶거나, 현재 노마드 생활 중인 사람들을 핵심 타겟으로 한다.

| 구분 | 특성 | 주요 니즈 |
|---|---|---|
| 예비 노마드 | 직장인·프리랜서, 원격근무 전환 준비 중 | 도시별 생활비·인터넷 비교, 시작 도시 추천 |
| 활동 중 노마드 | 개발자·디자이너·작가·마케터·크리에이터 | 현재 도시 커뮤니티, 코워킹·카페 정보 |
| 장기 이주 탐색자 | 지방 이주 고려 중, 재택 근무 정착 희망 | 생활비·주거·인프라, 지역 정착 가이드 |

### 2.2 사용자 공통 특성

- 컴퓨터·스마트폰 활용 능숙, 디지털 서비스에 익숙
- SNS(인스타그램·유튜브·X) 활발히 사용 — 바이럴 확산 채널
- 소프트웨어 개발·글쓰기·마케팅·디자인 등 장소 독립적 업무
- 정보 탐색에 적극적이며 커뮤니티 참여 의지 높음

## 3. 제안 해결책

한국 주요 도시별 노마드 생활 조건을 카드 형태로 시각화하고, 사용자들이 직접 평가·공유할 수 있는 플랫폼을 구축한다. 글로벌 레퍼런스인 Nomads.com의 핵심 구조를 참고하되, 한국 특화 지표와 UX를 적용한다.

### 3.1 핵심 가치 제안

- **한 곳에서** — 서울·부산·제주·강릉 등 주요 도시 조건을 카드 하나로 비교
- **한국 특화** — 원화 생활비, 카페 WiFi 밀도, 전셋값 등 글로벌 서비스에 없는 지표
- **신뢰할 수 있는 데이터** — 실제 거주자 크라우드소싱 평가 기반

## 4. 목표 및 성공 지표

### 4.1 런칭 목표 (첫 달)

| 지표 | 목표값 | 측정 방법 |
|---|---|---|
| MAU | 1,000명 이상 | GA 월간 활성 사용자 |
| 재방문율 | 30% 이상 | GA 재방문 세션 비율 |
| 평균 세션 시간 | 3분 이상 | GA 평균 세션 지속시간 |
| 도시 카드 클릭률 | 카드당 20% 이상 | 카드 클릭 / 노출 수 |
| 평가 등록 수 | 월 100건 이상 | 평가 제출 완료 이벤트 |

### 4.2 보조 지표

- SNS 공유 수 — 카드별 공유 버튼 클릭
- 검색 유입 비율 — 오가닉 트래픽 30% 이상
- 이탈율 — 홈 페이지 기준 60% 이하

## 5. 홈페이지 기능 명세

> 본 PRD는 홈페이지(도시 카드 목록 + 평가 시스템) 범위만 다룬다.  
> 커뮤니티·밋업·수익화 기능은 다음 단계 PRD로 분리한다.

### 5.1 도시 카드 목록

#### 5.1.1 지원 도시

- **1차 (출시)**: 서울 · 부산 · 제주 · 인천 · 대전 · 대구 · 광주
- **2차 (출시 후 추가)**: 춘천 · 강릉 · 전주 · 경주 · 여수 · 통영

#### 5.1.2 도시 카드 표시 항목

- 도시명 · 지역명 · 랭킹 번호 · 배지 (인기 / 자연 / 가성비 등)
- 썸네일 이미지
- 5가지 핵심 지표 점수 바 (5점 만점)
  - ★ 종합 점수 — 항목별 가중 평균
  - ◎ 인터넷 속도 — 실측 Mbps 기반
  - ₩ 생활비 — 월 원화 기준
  - ◈ 안전 — 사용자 평가
  - ♨ 카페 밀도 — 사용자 평가
- 월 생활비 범위 (만원 단위) · 코워킹스페이스 수
- 상세보기 버튼 · 저장(북마크) 버튼

#### 5.1.3 필터 & 정렬

- 지역 필터: 수도권 · 강원 · 충청 · 영남 · 호남 · 제주
- 생활비 범위 슬라이더 (만원 단위)
- 최소 인터넷 속도: 상관없음 / 100M / 500M / 1G 이상
- 코워킹스페이스 유무 필터
- 정렬: 종합점수순 · 생활비 낮은순 · 인터넷 빠른순 · 최신 평가순

### 5.2 도시 카드 상세 (펼침)

- 7가지 항목별 상세 점수 바
  - 종합 · 인터넷 · 생활비 · 안전 · 카페 밀도 · 교통 · 워라밸
- 실측 인터넷 속도 Mbps · 평가 건수 표시
- 월 생활비 범위
- 코워킹스페이스 수 · 카페 수
- 태그: `#빠른인터넷` `#가성비` `#바다근처` 등
- 리뷰 보기 링크 · 지도 보기 링크

### 5.3 사용자 평가 시스템

#### 5.3.1 평가 입력 항목

- 체류 기간: 1개월 미만 / 1~3개월 / 3개월 이상
- 방문 목적: 원격근무 / 프리랜서 / 여행 / 장기거주
- 종합 만족도: 별점 1~5
- 항목별 점수 (1~5): 인터넷 속도 · 생활비 · 카페/코워킹 · 대중교통 · 안전 · 외국인 친화도
- 한 줄 후기 (선택, 100자 이내)

#### 5.3.2 평가 권한

- 읽기: 비회원 포함 누구나
- 작성: 소셜 로그인 회원만 (카카오 · 네이버)
- 중복 방지: 동일 도시 재평가 30일 제한

## 7. 기술 스택

| 구분 | 기술 | 비고 |

| 백엔드 | Spring Boot 3, Java 17+ | 기존 사내 스택 유지 |

| 배포 | 미정 (AWS / Naver Cloud) | 추후 결정 |

## 8. 제약 및 전제 조건

### 8.1 범위 제한 (이번 PRD)

- 홈페이지(도시 카드 목록 + 평가)만 다룬다
- 커뮤니티·밋업·채용 보드·수익화는 다음 단계 PRD로 분리
- 도시 상세 페이지, 지도 뷰, 마이페이지는 후속 PRD

### 8.2 전제 조건

- 초기 도시 데이터(생활비·코워킹 수 등)는 운영팀이 수기 입력
- 실측 인터넷 속도는 사용자 제보 크라우드소싱으로 수집
- 소셜 로그인(카카오·네이버) API 키 사전 발급 필요
